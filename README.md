# it-case-studies
A collection of IT case studies solving network, hardware, cloud, software, security, and troubleshooting issues.

## Skills / Categories
### Software / Applications
End-user issues with desktop or cloud apps

### Networking / Connectivity
Network, Wi-Fi, VPN, DNS, or IP issues

### Windows / System Administration
OS-level issues, user accounts, policies

### Hardware / Peripherals
Physical device troubleshooting

### Cloud / Microsoft 365
Account management or cloud apps

### Security / Access
Account lockouts, MFA, encryption, permissions

### Automation / Scripting (optional but powerful)
Tasks automated with PowerShell or Python

### General / Customer Support
Soft skills, ticket handling, documentation
