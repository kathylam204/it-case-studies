# Case Study: OneDrive Cache Issue
## Summary
A friend encountered issues with OneDrive not syncing files properly across devices.
## Context
Service: OneDrive (Cloud Storage)
## Actions Taken
- Diagnosed corrupted cache in OneDrive.
- Guided the user to clear the cache via browser settings.
## Result
OneDrive resumed syncing without errors.
## Reflection
This case reinforces the importance of regularly clearing app data and cache to resolve syncing issues, particularly for cloud services.
